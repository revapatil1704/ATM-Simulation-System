# ATM-Simulation-System
ATM Simulation System is a simple Java console-based project that simulates basic ATM operations. The system allows users to securely log in using a PIN and perform actions like balance check, deposit, and withdrawal. This project is developed using basic Object-Oriented Programming (OOP) concepts and is suitable for beginners.


The user is required to enter a PIN for authentication.  
After successful login, the user can:
- Check account balance
- Deposit money
- Withdraw money
- Exit the session


## Features
- PIN verification
- Balance check
- Deposit money
- Withdraw money
- Simple menu-driven program


## Technologies Used
- Java
- OOP Concepts
- Console Input (Scanner)

## How to Run
1. Compile the program  
   `javac ATMSystem.java`
2. Run the program  
   `java ATMSystem`

## Project Structure
- ATMSystem.java (Main class)
